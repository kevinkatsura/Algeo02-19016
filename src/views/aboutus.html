About This Website

Overview

This website is created for fulfilling Aljabar Linier dan Geomerty course assignment. This website is basicly a very simple search engine which reads and filtering documents uploaded in local. There is the main search bar to make a search within the uploaded documents and there is also a section where user can upload another file to add to local. File upload only handle (.txt) extension. Every documents provided are all in Bahasa Indonesia, so any search which are not in Bahasa Indonesia may result in 0% similarity.

Search Engine with Cosine Similarity

Cosine similarity is a tool to find how close are texts with the query given. Cosine similarity is based on vector dot product and megnitude. The formula for cosine similarity follows:
cosine similarity = (dot product document vector and query vector) / ( magnitude document vector * magnitude query vector)
